Landslide SLIP Tool

Introduction

Summary

SLIP is used to automate the detection of landslides. A SLIP product is the result of filtering based on per-pixel changes in both soil moisture and vegetation in areas with high elevation gradients. All of which (with the exception of elevation gradients) can be computed using simple bandmath equations. Define a bounding box, time period, and data output options and press the submit button to run a task.

Input Options

The available options include the baseline type (average or composite) and the baseline length (number of acquisitions). When you submit a new task, the entire time stack of data from the specified bounding box and date range will be loaded and processed based on the input parameters.

Output Data

The output image will contain a mosaic of the baseline scenes with pixels that exhibit signs of change being highlighted in red. The mosaic without red highlighted pixels is also available in the results and output panels. The data product is available for download in both NetCDF and GeoTIFF format along with any images created.

Task Parameters

Data Selection

The data selection section contains options that affect the data and processes that are used to create the output product.

Baseline Method

The available options are average and composite. Selecting average will use the mean of previous scenes as a baseline, while selecting composite will create a most recent pixel mosaic for the baseline.

Baseline Length

The baseline length is the number of acquisitions that will be used to create the baseline. For example, selecting 5 will use the five scenes preceeding the scene that is being analyzed for the baseline.

Geospatial Bounds

These bounds will be use to filter the available data. For any given bounding box, each acquisition that is found between start and end date will be used to create the mosaic. The bounding box can either be drawn on the map by a single click, drag, and second click, or by entering valid values in the boxes.

Task Outputs

Output Image Display

The output image will contain a mosaic of the baseline scenes with pixels that exhibit signs of change being highlighted in red. The mosaic without red highlighted pixels is also available in the results and output panels. The data product is available for download in both NetCDF and GeoTIFF format along with any images created.

Managing Completed Tasks

In the results tab, selecting a task will load the list of scenes that contain SLIP classified pixels. From this panel, you are able to load the SLIP analysis of any single acquisition by clicking "Load this scene." To remove a task from the map display, press either delete or the hide button on the results tab.

Product Download

All products can be downloaded using the output tab. A GeoTIFF and NetCDF are available the result of the SLIP output, as well as the map image and the mosaic image.
