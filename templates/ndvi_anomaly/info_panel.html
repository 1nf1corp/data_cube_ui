NDVI Anomaly

Introduction

Summary

The NDVI Anomaly Tool compares the NDVI of a single acquisition to a baseline formed from the median of multiple scenes from the past. Months are selected to define a custom baseline period: Past scenes from the selected months are combined to form the median. This tool is based on the GEOGLAM Crop Monitor tool.

Input Options

The available input options include the geographic extent, the desired scene by acquisition date, and the baseline period in months. Please note that selecting a geographic extent that has no intersection with the selected scene will return no results.

Output Data

Four output images will be produced: The NDVI difference, the NDVI percentage change, the NDVI of the selected scene, and a true color image of the selected scene. A GeoTIFF and NetCDF file is also produced for the NDVI based products.

Task Parameters

Data Selection

The Data Selection subsection includes options used to form the baseline.

Baseline Period

Choose one or more months to form a baseline comprised of past scenes that occur within those months. To produce more meaningful results, choose baseline months that correspond with the season of your selected scene.

Geospatial Bounds

These bounds will be use to filter the available data. The bounding box can either be drawn on the map by a single click, drag, and second click, or by entering valid values in the boxes. The scene selection option determines what acquisition is compared to the baseline. This is a single scene operation, so expect some missing data due to banding or no-data.

Task Outputs

Output Image Display

The default displayed image is the NDVI difference, or the baseline NDVI subtracted from the scene NDVI. The percentage change between the baseline and the scene is also available, along with the scene NDVI and the scene image. The color scales for these images can be found under the color scale header on this panel.

Managing Completed Tasks

Various metadata is available under the results tab. The displayed image is selected here, and a listing of all scene metadata and metadata from all baseline scenes can be seen here. To remove a task from the map display, press either delete or the hide button on the results tab.

Product Download

All display products can be downloaded from the output tab. A NetCDF and GeoTIFF is avaible containing the NDVI difference, NDVI percent change, and scene NDVI.
